= AsciiDocでブログを始める

== はじめに

このブログはAsciiDocで書かれています。
Markdownと比べて、以下のような利点があります。

* テーブルが書きやすい
* admonition（NOTE, TIP, WARNING等）が標準サポート
* includeディレクティブでファイル分割が可能

== コードブロックの例

[source,go]
----
package main

import "fmt"

func main() {
    fmt.Println("Hello, AsciiDoc!")
}
----

== Admonitionの例

NOTE: AsciiDocではNOTE, TIP, IMPORTANT, WARNING, CAUTIONが使えます。

TIP: `source-highlighter` に `rouge` を指定するとシンタックスハイライトが有効になります。
