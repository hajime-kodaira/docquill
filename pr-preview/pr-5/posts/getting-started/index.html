AsciiDocでブログを始める

2025-01-01 ブログ

1. はじめに

このブログはAsciiDocで書かれています。 Markdownと比べて、以下のような利点があります。

  • テーブルが書きやすい

  • admonition（NOTE, TIP, WARNING等）が標準サポート

  • includeディレクティブでファイル分割が可能

2. コードブロックの例

package main

import "fmt"

func main() {
    fmt.Println("Hello, AsciiDoc!")
}

Note
AsciiDocではNOTE, TIP, IMPORTANT, WARNING, CAUTIONが使えます。
Tip
source-highlighter に rouge を指定するとシンタックスハイライトが有効になります。
