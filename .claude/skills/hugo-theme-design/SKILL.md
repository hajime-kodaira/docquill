# Hugo テーマデザインスキル

## デザイン方針

「AIっぽいダサさ」を避け、プロダクション品質のUIを構築する。

### 避けるべきパターン
- Inter, Roboto, Arial などの汎用フォント
- 紫グラデーション on 白背景
- 均一な角丸のカード
- 予測可能なレイアウト

### 追求すべきもの
- 個性のあるフォントペアリング（見出し + 本文で異なるフォント）
- 意図のあるカラーパレット（CSS Variablesで一元管理）
- タイポグラフィ主体のデザイン（テックブログに相応しい読みやすさ）
- ダークモード時の色設計（単なる反転ではなく、専用の配色）

## テーマカラー定義

CSS Variables で管理し、ダークモード切り替えを容易にする。

## レイアウト要件

### 必須テンプレート
- `baseof.html` — 共通レイアウト（head, header, footer）
- `index.html` — トップページ（最新記事一覧）
- `single.html` — 記事ページ（目次 + 本文 + タグ）
- `list.html` — セクション一覧（ページネーション付き）
- `taxonomy/list.html` — 全タグ/カテゴリ一覧
- `taxonomy/term.html` — 個別タグ/カテゴリの記事一覧

### レスポンシブブレークポイント
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### コードブロック
- 横スクロール対応（`overflow-x: auto`）
- 行番号はオプショナル
