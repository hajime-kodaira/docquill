1. Admonitionの全種類

Note
ノートです。補足情報を表します。
Tip
ヒントです。便利な使い方を示します。
Important
重要事項です。読者に強く伝えたい内容です。
Warning
警告です。注意を喚起します。
Caution
注意です。慎重な取り扱いが必要です。

3. ソースコードブロック

Pythonの例
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
Bashの例
#!/bin/bash
for i in {1..5}; do
  echo "iteration $i"
done
